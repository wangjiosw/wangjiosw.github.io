# 1. 问题描述

# 2. 错误的原因

Hadoop jline 版本和 hive 的 jline 不一致

# 3. 解决方法

删除 your_install_path/hadoop/share/hadoop/yarn/lib 目录下的 jline 包，然后把 hive 里面的 jline 包拷过来。
